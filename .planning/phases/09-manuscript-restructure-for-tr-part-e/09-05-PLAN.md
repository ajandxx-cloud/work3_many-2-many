# Plan 09-05: Tables, Figures, Limitations, and Managerial Insights Plan

<objective>
Create the Phase 9 table/figure plan that assigns every paper-facing display to
main text or appendix/supplement, and converts the current policy section into
limitations-first, simulation-based managerial insights with explicit boundary
conditions.
</objective>

<must_haves>
- D-10: Primary table contains only formal main evidence matrix and approved metrics.
- D-11: Matched-coverage and fixed accepted-set controls are compact robustness summaries with full details in appendix/supplement.
- D-12: Algorithm diagnostics are scoped and mostly appendix/supplement detail.
- D-13: Equity and passenger-type results are a main-text trade-off summary with details in appendix/supplement.
- D-14: No ambiguous `vkm_per_trip` language.
- D-15: Rename old policy section to `Managerial Insights and Boundary Conditions`.
- D-16: Rewrite R1-R5 as conditional insight bullets with condition, supported insight, applicability boundary, and limitation.
- D-17: Beijing/city-density language remains `Beijing-inspired synthetic scenario`.
- D-18: Managerial insights answer when bidirectional meeting-point design is worth considering, not parameter prescriptions.
</must_haves>

<truths>
- Current figures and scripts live under `manuscript/figures/` and `manuscript/figures/scripts/`.
- Current policy section contains R1-R5 recommendations and a policy map that read more prescriptive than the evidence gate allows.
- Elsevier requires figure/table order, captions, and accurate supplementary material.
- Phase 8 supported claims are absent, so final captions and insight text require placeholders.
</truths>

<threat_model>
| Threat | Severity | Mitigation |
|---|---|---|
| Old figures imply unsupported headline evidence | high | Require Phase 8 support status for every main display |
| Detailed diagnostics crowd the main text | medium | Allocate complete diagnostics to appendix/supplement |
| Managerial insights become universal policy prescriptions | high | Use condition/insight/boundary/limitation template |
| Figure artwork violates reproducibility expectations | low | Require generated plots from scripts/data and avoid AI-created artwork |
| Captions use forbidden metric names | high | Add caption metric-vocabulary checklist |
</threat_model>
