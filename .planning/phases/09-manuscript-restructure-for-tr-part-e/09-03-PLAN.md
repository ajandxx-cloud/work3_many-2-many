# Plan 09-03: Introduction and Literature Positioning Plan

<objective>
Create a revised introduction plan that opens with the reviewer-facing evidence
gap, uses the three approved research questions, orders contributions around
evidence discipline before method/results/insight, and gives literature
positioning rules that avoid broad first/only claims.
</objective>

<must_haves>
- D-01: Manuscript storyline is evidence-chain reconstruction.
- D-02: Introduction opens with the fair-evidence problem.
- D-03: Tone is cautious and conditional.
- D-04: Contributions are ordered as evidence discipline, model/framework, experimental findings, managerial implications.
- D-05: Introduction supports the new TR-E evidence-chain structure.
- D-19: Fair-evidence problem leads abstract and introduction.
- D-20: Core contribution is an evidence-chain framework.
- D-21: Introduction uses the three approved research questions.
- D-22: Algorithm diagnostics are not standalone research questions.
</must_haves>

<truths>
- Current introduction says the present paper fills a broad combined gap and includes old effect sizes.
- Phase 1 novelty positioning forbids broad first/only language unless precisely verified.
- The introduction should set up Phase 8 claim gating rather than promise final results prematurely.
</truths>

<threat_model>
| Threat | Severity | Mitigation |
|---|---|---|
| Literature positioning revives unverified novelty claims | high | Include allowed/risky/forbidden wording table sourced from Phase 1 |
| Algorithm diagnostics become a headline contribution | medium | State diagnostics are methodological support, not a research question |
| Introduction promises unsupported results | high | Use Phase 8 placeholders for result sentences |
| Contribution list leads with method superiority | medium | Force contribution order from D-04 |
</threat_model>
