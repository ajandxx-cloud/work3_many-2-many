Create analysis/sensitivity.py with two public functions: sweep_walking_tolerance() and sweep_fleet_size().

sweep_walking_tolerance():
- RHO_VALUES = [200, 300, 400, 500, 700, 1000]  (meters, per POLICY-02)
- For each rho in RHO_VALUES:
  - Temporarily set experiments.config.RHO_P = rho and experiments.config.RHO_D = rho
  - Run generate_synthetic(200, 15, 42) to get scenario
  - Run FullModel().run(scenario) and DoorToDoor().run(scenario)
  - Compute metrics for each; append row with density_tier="standard"
- Restore original RHO_P / RHO_D after each iteration (use try/finally)
- City tier rows (POLICY-06): after the main sweep, run FullModel only on:
  - generate_synthetic(100, 10, 42) with rho=500 (midpoint), density_tier="low"
  - generate_synthetic(400, 20, 42) with rho=500 (midpoint), density_tier="high"
- Return list of row dicts; also write to results/sensitivity_walk.csv using csv.DictWriter

sweep_fleet_size():
- FLEET_VALUES = [5, 10, 15, 20, 25, 30]  (per POLICY-03)
- n_requests = 200 fixed
- For each n_vehicles in FLEET_VALUES:
  - Run generate_synthetic(200, n_vehicles, 42)
  - Run FullModel().run(scenario) and DoorToDoor().run(scenario)
  - Compute metrics; append row with keys: n_vehicles, variant, acceptance_rate, vehicle_km, avg_wait
- Write to results/sensitivity_fleet.csv using csv.DictWriter
- Return list of row dicts

Important: variants.py imports RHO_P/RHO_D from experiments.config at module load time into local variables. To override them per-run, pass rho_p and rho_d as constructor arguments to FullModel and DoorToDoor if those constructors accept them — otherwise, patch experiments.config.RHO_P and experiments.config.RHO_D AND re-import the constants inside the variant's run() method. Check variants.py to see which approach is used. If variants read from config at call time (not import time), patching config is sufficient. If they cache at import time, add optional rho_p/rho_d parameters to the variant constructors.

Create results/ directory if it does not exist (os.makedirs("results", exist_ok=True)).

At the bottom of sensitivity.py, add:
