Algorithm:
1. SEEDS = [42, 43, 44] (from experiments.config)
2. For each seed, run generate_synthetic(200, 15, seed) then FullModel().run(scenario)
3. Collect all PassengerRecord objects across the 3 runs
4. Group records by passenger_type: "price_sensitive", "time_sensitive", "walk_sensitive"
5. For each type, compute:
   - acceptance_rate = accepted_count / total_count (across all 3 seeds combined)
   - avg_wait = mean(wait_time) for accepted records only (seconds)
   - avg_walk = mean(pickup_walk + dropoff_walk) for accepted records only (meters)
   - avg_ivt = mean(ivt) for accepted records only (seconds)
6. Compute Gini coefficient of the three acceptance_rate values using the same _gini() formula from experiments/metrics.py (copy the formula inline — do not import private function):
   ```python
   def _gini_three(values):
       arr = sorted(values)
       n = len(arr)
       s = sum(arr)
       if s == 0:
           return 0.0
       return (2 * sum((i+1)*v for i, v in enumerate(arr)) / (n * s)) - (n+1)/n
   ```
7. Build rows list: one dict per type with all 5 metrics + gini_acceptance (same value repeated)
8. Write to results/equity_table.csv using csv.DictWriter with fieldnames:
   passenger_type, acceptance_rate, avg_wait, avg_walk, avg_ivt, gini_acceptance
9. Return {"rows": rows, "gini": gini_value}
