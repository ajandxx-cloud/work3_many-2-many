Create `src/drt/types.py`:
- Use `@dataclass` from `dataclasses` stdlib (no Pydantic)
- `Request`: fields `id: str`, `origin: tuple[float, float]`, `destination: tuple[float, float]`, `earliest: float`, `latest: float`, `max_ride_time: float`
- `Vehicle`: fields `id: str`, `capacity: int`, `max_route_duration: float`, `current_position: tuple[float, float]`, `current_time: float`
- `MeetingPoint`: fields `id: str`, `coords: tuple[float, float]`
- `Bundle`: fields `request_id: str`, `pickup_mp: MeetingPoint`, `dropoff_mp: MeetingPoint`, `departure_time: float`, `price: float`
- `Route`: fields `vehicle_id: str`, `stops: list` (each stop is a tuple of (MeetingPoint, float) for scheduled time)
- `PassengerType`: fields `name: str`, `beta_walk: float`, `beta_wait: float`, `beta_ivt: float`, `beta_price: float`

Create `src/drt/choice.py`:
- Import `math`, `Request`, `Bundle`, `MeetingPoint`, `PassengerType` from `.types`
- Helper `euclidean(a, b)` — Euclidean distance between two (x,y) tuples
- `mnl_utility(bundle: Bundle, request: Request, ptype: PassengerType, current_time: float) -> float`:
  - walk_dist = euclidean(request.origin, bundle.pickup_mp.coords) + euclidean(bundle.dropoff_mp.coords, request.destination)
  - wait_time = bundle.departure_time - current_time
  - ivt = euclidean(bundle.pickup_mp.coords, bundle.dropoff_mp.coords)  # proxy for in-vehicle time
  - return ptype.beta_walk * walk_dist + ptype.beta_wait * wait_time + ptype.beta_ivt * ivt + ptype.beta_price * bundle.price
  - (All β are negative for disutility attributes — callers set negative values)
- `choice_probability(bundles: list[Bundle], request: Request, ptype: PassengerType, current_time: float) -> dict`:
  - Compute U_rb for each bundle using mnl_utility
  - Outside option U_r0 = 0.0
  - exp_sum = exp(0.0) + sum(exp(U_rb) for each bundle)
  - Return {bundle: exp(U_rb)/exp_sum for each bundle, None: exp(0.0)/exp_sum}
  - Keys: Bundle objects + None (outside option)
  </action>
