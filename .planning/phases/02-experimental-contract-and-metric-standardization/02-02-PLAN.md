# 02-02 Plan: Metric Definitions and Coverage-Confounding Controls

<objective>
Create standalone metric definitions and coverage-confounding controls that prevent ambiguous denominators, hidden rejection mechanisms, and coverage-driven vehicle-km claims from becoming final evidence.
</objective>

## Scope

In scope:

- Define formulas, denominators, units, valid ranges, and interpretations for Phase 2 metrics.
- Separate served share, behavioral acceptance, choice rejection, feasibility rejection, deterministic inserted share, total vehicle-km, vehicle-km per served trip, and vehicle-km per original request.
- Define unconstrained behavioral, matched-coverage, and fixed accepted-set evidence roles.
- Forbid high-risk metric and claim language.

Out of scope:

- Implementing new metric code.
- Running matched-coverage or fixed accepted-set experiments.
- Calibrating passenger utility parameters.
