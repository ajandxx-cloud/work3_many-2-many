# Plan 10-02: Reviewer Reproduction Guide and Artifact Index

<objective>
Create the reviewer-facing reproduction guide that records environment setup,
reproduction entry points, manuscript build commands, table/figure generation
chains, and the human-readable final artifact index while preserving the
blocked state caused by missing Phase 6 and Phase 8 evidence.
</objective>

<must_haves>
- D-02: `10_REPRODUCIBILITY.md` must be created even when upstream evidence is blocked.
- D-05: Reviewer-facing entry points prioritize final main tables/figures, Phase 6 formal artifacts, Phase 8 claim gate, and manuscript build.
- D-06: Pilot and legacy artifacts may appear only as readiness, provenance, or legacy.
- D-07: Diagnostic and supplementary commands must be layered by `critical robustness`, `supplementary diagnostic`, and `legacy diagnostic`.
- D-08: Manuscript build reproducibility includes LaTeX/PDF build, table/figure generation chain, input data paths, output PDF paths, output figure/table paths, and script paths.
- D-11: Record code revision and dependency commands; checksums and dependency snapshots are improvements, not blockers.
- D-16: The final artifact index appears as a human-readable section in `10_REPRODUCIBILITY.md`; `10_RESULT_MANIFEST.md` remains the structured table.
</must_haves>

<truths>
- `10_RESULT_MANIFEST.md` is the structured source for artifact roles and statuses.
- `manuscript/main.tex`, `manuscript/sections/`, `manuscript/figures/scripts/`, and `manuscript/figures/` are the current manuscript build and display surface.
- Phase 9 requires old numerical claims and captions to remain placeholders until Phase 8 claim support exists.
- Current figure scripts are reproducible assets but may encode legacy titles or values that must be regenerated after Phase 6/8 support.
</truths>

<threat_model>
| Threat | Severity | Mitigation |
|---|---|---|
| Reproduction guide implies final results can be regenerated despite missing prerequisites | high | Put `Blocked: prerequisites missing` in the guide status and entry-point notes |
| Reviewer commands mix final evidence with pilot or legacy outputs | high | Separate command groups by evidence role |
| Manuscript build steps omit table/figure provenance | medium | List LaTeX, figure scripts, input data, and output paths together |
| Environment setup fails in a fresh checkout | medium | Include install, dependency inspection, and test commands |
| Human artifact index diverges from structured manifest | medium | Direct executors to build the index from `10_RESULT_MANIFEST.md` rows |
</threat_model>
