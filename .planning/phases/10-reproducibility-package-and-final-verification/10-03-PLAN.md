# Plan 10-03: Final Verification Gate Matrix

<objective>
Create the final Phase 10 verification artifact that checks prerequisites,
artifact manifests, reproduction entry points, tables, figures, and manuscript
claims, and that fails closed with blocked/pending statuses until Phase 6 formal
evidence and Phase 8 claim-gate files exist.
</objective>

<must_haves>
- D-01: Phase 10 must fail closed if Phase 6 formal results/artifacts or Phase 8 claim-gate files are missing.
- D-03: The Phase 6 formal report and three Phase 8 claim-gate files are hard blockers.
- D-04: The final blocked wording is `Blocked: prerequisites missing`.
- D-12: When Phase 8 is missing, artifact `claim_link` remains `pending Phase 8` and artifacts cannot support final claims.
- D-13: `10_FINAL_VERIFICATION.md` must use a gate matrix listing prerequisites, artifacts, tables/figures, claims, status, evidence paths, and blockers.
- D-14: Verification statuses are exactly `Pass`, `Pending`, `Blocked`, and `Not final evidence`.
- D-15: Claim verification operates at each final manuscript claim level; every final claim needs Phase 8 evidence, result artifacts, and table/figure links.
- D-16: The final artifact index lives in `10_REPRODUCIBILITY.md`, while this artifact verifies it.
</must_haves>

<truths>
- `10_RESULT_MANIFEST.md` and `10_REPRODUCIBILITY.md` must exist before this final verification report is useful.
- Phase 9 locked all final manuscript claims behind Phase 8 supported/unsupported claim artifacts.
- Current manuscript sections may contain legacy prose, but Phase 10 should document verification state rather than rewriting final manuscript claims.
- Missing upstream evidence should produce a useful blocked report, not an absent report.
</truths>

<threat_model>
| Threat | Severity | Mitigation |
|---|---|---|
| Final verification is marked passed despite missing Phase 6 or Phase 8 evidence | high | Require top-level status `Blocked: prerequisites missing` while hard blockers are absent |
| Claim verification is performed only at section level | high | Require one row per final manuscript claim or placeholder claim item |
| Tables and figures are verified without data provenance | high | Link each display to manifest row, source command, input path, output path, and Phase 8 status |
| Artifact index exists but is not checked | medium | Add a gate row for `10_REPRODUCIBILITY.md` final artifact index |
| Legacy manuscript claims remain invisible | medium | Add a scan checklist for claim-like statements in manuscript sections |
</threat_model>
