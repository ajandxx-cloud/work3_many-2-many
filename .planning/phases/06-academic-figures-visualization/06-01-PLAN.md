## FIG-03: Algorithm flowchart

Write figures/scripts/fig03_algorithm.py producing figures/fig03_algorithm.pdf and .png.

Figure spec (single-column, 88mm × 160mm, 300 dpi):
- figsize=(3.46, 6.30) inches, dpi=300, font serif 9pt
- Boxes: rectangles for process steps '#4878CF'; diamonds for decisions '#D65F5F'; rounded rect for start/end '#6ACC65'

Flowchart nodes (top to bottom):
1. START oval: "Request r arrives"
2. Process box: "Generate candidates\nM_r^P, M_r^D (walking radius ρ filter)"
3. Process box: "Enumerate bundles (m^P, m^D)\nfor each vehicle v, positions (π^P, π^D)"
4. Decision diamond: "Feasible insertion\nexists?"
   - No branch → box "Reject request (z_r=0)" → END
   - Yes branch → continue
5. Process box: "Select best bundle b*\n(min incremental cost ΔC)"
6. Process box: "Compute MNL probability\nP_{rb*}^k (Eq. choice-prob)"
7. Decision diamond: "Passenger accepts?\n(Bernoulli sample z_r)"
   - No branch → box "Discard x_r\nroutes unchanged" → END
   - Yes branch → continue
8. Process box: "Insert m_r^P at π_r^P,\nm_r^D at π_r^D in route σ_{v_r}"
9. Decision diamond: "Rolling horizon\ntrigger? (t mod Δ = 0)"
   - No branch → END
   - Yes branch → box "ALNS re-opt on\nR^act(t_now) over [t, t+H]" → END
10. END oval: "Dispatch updated routes"

Use matplotlib.patches.FancyBboxPatch for boxes, FancyArrowPatch for arrows, and a custom diamond shape (Polygon with 4 vertices) for decisions.
