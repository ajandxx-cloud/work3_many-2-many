# 05-02 Plan: Coverage-Control and Fixed Accepted-Set Smokes

<objective>
Add Phase 5 readiness diagnostics for matched coverage and fixed accepted sets, with durable diagnostic rows and blocking status only where the Phase 5 context requires it.
</objective>

## Scope

In scope:

- Add pilot-scale matched-coverage diagnostic rows.
- Convert matched-coverage tolerance failure into a blocking gate result.
- Add a minimal fixed accepted-set smoke using one routing diagnostic.
- Preserve no-Gurobi behavior as non-blocking durable diagnostic output.

Out of scope:

- Broad ALNS/MILP benchmarking.
- Formal evidence, confidence intervals, or method-superiority claims.
- Case-study or Beijing-inspired diagnostics.
