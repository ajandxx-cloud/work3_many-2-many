# 05-03 Plan: Pilot Execution, Gate Report, and Bug Ledger

<objective>
Run the Phase 5 pilot gate, validate all pilot artifacts, write the no-claims pilot report, and maintain a structured bug ledger that blocks Phase 6 until core readiness failures are fixed and rerun.
</objective>

## Scope

In scope:

- Execute the Phase 5 pilot harness and diagnostics.
- Run validators against persisted artifacts.
- Generate diagnostic plots that summarize status/range/missingness only.
- Write `05_PILOT_RESULTS.md` as a gate report.
- Maintain `05_BUG_LEDGER.csv`.
- Update `.planning/STATE.md` to show Phase 5 execution status after the report is written.

Out of scope:

- Formal Phase 6 experiment runs.
- Manuscript-ready plots or final claims.
- Case-study smoke runs.
