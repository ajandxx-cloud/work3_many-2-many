Define abstract base:
```python
import time, math
from abc import ABC, abstractmethod
from dataclasses import dataclass
from src.drt.types import PRICE_SENSITIVE, TIME_SENSITIVE, WALK_SENSITIVE

PASSENGER_TYPES = [PRICE_SENSITIVE, TIME_SENSITIVE, WALK_SENSITIVE]
TRAVEL_SPEED = 8.33  # m/s ≈ 30 km/h

class BaseVariant(ABC):
    name: str

    def run(self, scenario) -> SimulationResult:
        t0 = time.perf_counter()
        state = self._solve(scenario)
        cpu_time = time.perf_counter() - t0
        records = self._build_records(scenario, state)
        vehicle_km = self._total_vehicle_km(state, scenario.vehicles)
        return SimulationResult(records=records, total_vehicle_km=vehicle_km, cpu_time=cpu_time)

    @abstractmethod
    def _solve(self, scenario) -> ALNSState: ...

    def _build_records(self, scenario, state) -> list[PassengerRecord]: ...
    def _total_vehicle_km(self, state, vehicles) -> float: ...
    def _euclidean(self, a, b) -> float: return math.hypot(a[0]-b[0], a[1]-b[1])
```

`_build_records` logic:
- For each request in scenario.requests:
  - Determine if accepted (request.id not in state.rejected and not in state.unassigned)
  - Assign passenger_type by cycling through PASSENGER_TYPES by request index (index % 3)
  - If accepted: find the stop in the route, compute wait_time, pickup_walk, dropoff_walk, ivt
  - direct_time = euclidean(origin, destination) / TRAVEL_SPEED
  - total_disutility = ptype.beta_walk * (pickup_walk+dropoff_walk) + ptype.beta_wait * wait_time + ptype.beta_ivt * ivt
  - If rejected: accepted=False, all times=0

`DoorToDoor`:
- _solve: use greedy_insertion with meeting_points=[] (empty list)
- Since no meeting points, pickup stop = request.origin, dropoff stop = request.destination
- pickup_walk = 0, dropoff_walk = 0

`SingleSidedPickup`:
- _solve: use greedy_insertion with full meeting_points for pickup, but force dropoff = destination
- Implementation: pass meeting_points for pickup candidates, rho_d=0 (forces dropoff at destination)
